# 项目如何来学习
link-flow项目的特点是业务和轮子相结合。既有着 **控制微服务流量的业务功能**，又有着 **对微服务框架进行定制增强的轮子功能**。实习和社招都能够用到！

再加上高并发业务的大麦项目，面试可以说是 **双剑合璧** 了。

为了避免有烂大街的风险，link-flow项目的代码将 **不再实行开源** 的方案，只有加入星球的小伙伴可以申请项目的权限，这也是为了更好的维护星球小伙伴的利益。
