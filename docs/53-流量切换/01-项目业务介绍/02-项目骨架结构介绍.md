# 项目骨架结构介绍

+ **link-flow-base-frame-starter**    基础核心功能模块
+ **link-flow-commons**    公共相关功能模块
    - **link-flow-apollo-common-starter**    apollo配置中心功能公共模块
    - **link-flow-common-starter**    公共功能模块
    - **link-flow-nacos-common-starter**    nacos配置中心功能公共模块
+ **link-flow-config**    配置中心功能模块
    - **link-flow-config-apollo-starter**    apollo配置中心功能模块
    - **link-flow-config-nacos-starter**    nacos配置中心功能模块
    - **link-flow-config-starter**    配置中心抽象模块
+ **link-flow-register**    注册中心功能模块
    - **link-flow-register-eureka-starter**    eureka注册中心功能模块
    - **link-flow-register-nacos-starter**    nacos注册中心功能模块
+ **link-flow-test**    测试功能模块
    - **link-flow-gateway-test**    gateway网关功能测试模块
    - **link-flow-service-test**    业务服务功能测试模块
+ **link-flow-work**    核心功能模块
    - **link-flow-work-gateway-starter**    gateway网关功能模块
    - **link-flow-work-hystrix-starter**    hystrix熔断功能模块
    - **link-flow-work-service-starter**    业务服务功能模块
    - **link-flow-work-starter**    功能抽象模块
