# 为你提供全方位的服务

## 为什么创办知识星球？

对于现在的 Java 程序员来说，无论是面试需要还是技能的提升，仅仅是八股文是不够的，还需要结合项目，而大部分的程序员所做的项目基本上可以说是平平无奇，
好不容易在网上找个优秀一点的项目，还已经都烂大街了

还有学生和刚参加工作的这类人，一开始对于上手工作是不太适应的，需要有人来指导，但是却没有地方找人问

基于以上的原因，本人想提供给大家技术讲解和项目实战的服务，但如果一直为爱发电，没有收入的情况下，坚持下去的动力和产出的质量肯定会下降的，毕竟人人都是要吃饭的。

为了让自己保证动力来服务大家，并且听取大家的意见，本人决定创办了知识星球社区 **JavaUp 项目实战&技术讲解**

本星球的特点就是通过 <font size="5" color="#5575f6"> **知识理论** </font>+<font size="5" color="#5575f6"> **项目实践** </font> 的特点来帮助小伙伴来学习，因为这样学习起来才会 “事半功倍”，目前推出了 大麦网 主打高并发实战的项目，
为了解决高并发问题设计了多种落地解决方案。

## 小伙伴的疑惑

### 学生人群

无论是正在大学中或者是培训机构中的学生，其实对项目的需求更高，可以这么说能不能决定你能正式工作的因素除了学历外就是项目了，既然学历已经成了定局，那么最能提高竞争力的就是 项目，如果能在简历和面试中已经有了比较出色的项目经验，那么对于面试官来说绝对是必杀技！

不用担心怕自己看不懂项目，项目的文档讲解非常的详细，将从 0 到 1 完整的过程都进行了讲解，想尽一切办法就是为了方便大家更好的理解。学生可根据自己目前学习的进度来跳转到对应的目录来学习

### 工作了几年的人群

对于这些人群，学习项目更是必要的，通过项目的讲解能学习到项目的架构设计、设计模式、高并发解决方案，来让自己的技术能力得到提高，并且让自己的简历和面试中通过此项目来增加个人亮点
