# 项目骨架结构介绍

+ **dock-data-center-agility-data-source-framework**    动态数据源功能

+ **dock-data-center-collect-data-service**    数据中台核心服务

+ **dock-data-center-common**    公共功能

+ **dock-data-center-extension-common**    公共的额外扩展功能

+ **dock-data-center-id-generator-framework**    分布式id功能

+ **dock-data-center-parameter**    核心参数
    - **dock-data-center-collect-parameter**    收集数据服务的参数

+ **dock-data-center-record-service**    数据记录服务

+ **dock-data-center-redis-tool-framework**    redis工具组件
  - **dock-data-center-redis-common-framework**    redis-common封装
  - **dock-data-center-redis-framework**    redis封装

+ **dock-data-center-redisson-framework**    redisson封装
  - **dock-data-center-redisson-common-framework**    公共配置
  - **dock-data-center-repeat-execute-limit-framework**    防重复执行幂等性工具
  - **dock-data-center-service-lock-framework**    分布式锁工具
+ **dock-data-center-thread-pool-framework**    线程池封装
