# 分库分表二次扩容方案

## 为何需要二次扩容

当使用基因法进行分库分表后，后续的业务增长可能会导致单表数据量再次超出预设的极限容量，从而影响系统性能和稳定性。因此，设计合理的二次扩容方案是很有必要。

本文将详细讲解如何在已经使用基因法分库分表的基础上，进行二次扩容的设计思路和实施步骤。
