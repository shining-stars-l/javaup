# 申请AI模型调用

## 申请DeepSeek
### 创建api key
#### 进入DeepSeek官网地址：[https://www.deepseek.com/](https://www.deepseek.com/)，然后点击右上角API开放平台

![](/img/damai-ai/申请AI模型调用/DeepSeek/1.png)

#### 登录进去
![](/img/damai-ai/申请AI模型调用/DeepSeek/2.png)

#### 选择左边的API keys，进行创建API key
![](/img/damai-ai/申请AI模型调用/DeepSeek/3.png)

![](/img/damai-ai/申请AI模型调用/DeepSeek/4.png)

### 充值
创建好api key后，并不能直接使用，需要充值才可以，也不需要充太多钱，只需**1元钱**就够我们用的了

#### 点击左边的充值后，需要先进行 实名认证

![](/img/damai-ai/申请AI模型调用/DeepSeek/5.png)

#### 选择 在线充值，支付金额选择 自定义，充值1元，然后选择支付宝或者微信进行支付就可以了

![](/img/damai-ai/申请AI模型调用/DeepSeek/6.png)


可以看下详细的价格介绍：[点击这里](https://api-docs.deepseek.com/zh-cn/quick_start/pricing)

### 项目配置DeepSeek的API-key
在 application.yaml 中配置DeepSeek的API-key，把deepseek的key替换成你创建的真正的API-key

![](/img/damai-ai/申请AI模型调用/DeepSeek/7.png)
